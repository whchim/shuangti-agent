# 领域知识约束强化

**日期**: 2026-06-26

## 修改内容

### 1. SYSTEM_PROMPT 强化
- 明确定义"可以回答"和"严格禁止"的问题范围
- 新增"拒绝越界"规则，非双体领域问题礼貌拒答
- 影响文件: `app/rag/chain.py`

### 2. ChromaDB 相似度阈值过滤
- RAG_SIMILARITY_THRESHOLD = 0.35，过滤低相关文档
- hybrid 模式下无高质量匹配时自动降级联网搜索
- 影响文件: `app/service/chat_service.py`、`app/api/chat.py`

### 3. 联网搜索添加领域前缀
- 搜索 query 前自动拼接"双体软件精英产业学院"
- 影响文件: `app/service/chat_service.py`、`app/api/chat.py`

### 4. 无结果时统一拒答
- 知识库 + 联网搜索均无结果时，直接返回专属拒答文案
- 不在 LLM 中凑回答，从源头阻断越界回答
- 影响文件: `app/service/chat_service.py`、`app/api/chat.py`
